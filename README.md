# Портфолио — Егор Толстых

Frontend-портфолио на чистом HTML, CSS и JavaScript.

**Сайт:** [egor-tolstykh-portfolio.vercel.app](https://egor-tolstykh-portfolio.vercel.app)

## Страницы

- **Главная** — `index.html`
- **Портфолио** — `portfolio.html`
- **Навыки** — `skills.html`
- **Контакты** — `contact.html`

## Локальный запуск

```bash
npx serve .
```

## Деплой на Vercel

1. Подключите репозиторий на [vercel.com](https://vercel.com)
2. Deploy — настройки подхватятся из `vercel.json`

HTML5 · CSS3 · JavaScript · GSAP · Swiper.js · Flexbox · Grid · БЭМ
